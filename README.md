# ClothesStore.API
This project consists in the creation of an Api in asp net core using clean architecure.
The purpose is to create an API for online clothing.
Using the following tech stack
* Asp Net Core 6
* SQRS
* Medietr
* Dependency Injeciton
* MSSQL
* JWT
* Auto mapper
* Fluent validator
* Etherneal email
* Stripe
